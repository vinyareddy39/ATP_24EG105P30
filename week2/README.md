# Week 2: Advanced JavaScript

This week, we learn more advanced JavaScript topics. We explore how to organize code using Classes and Modules, how to talk to other servers using APIs, and how to work with timers. We also look at a simple introduction to MongoDB databases.

**Contents:**
- APIs
- Advanced Array Operations
- Modules
- Classes
- Spread and Rest operators
- Timers
- Shallow and Deep copy concepts
- MongoDB basics
